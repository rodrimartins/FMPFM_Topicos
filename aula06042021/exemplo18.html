Pop, push, shift, Unshift, sort, reverse

pop()

O método pop() remove o último elemento do array

shift()

O método shift() remove o primeiro elemento do array

push()

O método push() adiciona um elemento no array

Unshift

O método unshift adiciona no inicio um novo elemento no array

sort() e reverse()

O método sort() ordena um array

O método reverse() classifica de maneira decrescente um array
